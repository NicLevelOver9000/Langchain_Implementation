# Langchain_Implementation
Basic RAG Implementation using Langchain.
Using gemini gemini-2.0-flash llm model and HuggingFace model sentence-transformers/all-MiniLM-L6-v2 for embedding

No credit information or billing account required.


#How to run :

-> python -m venv venv
-> venv\Scripts\activate.bat in cmd terminal

-> pip install -r requirements.txt
-> python main.py
